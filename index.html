What's Trivial_i1b about?

Trivial_i1b is a project for Software Architecture course in Computer Engineering degree.

The main goal of the project is to develop a Trivial game, following the indications gaved to us. It is based on the Trivial Pursuit game, the typical game based on a board and with different questions that belong into different categories to be answered by the players.

The application is developed in Java, using Eclipse, making use of the Maven plugin and different testing tools to test the correct performance of the game.

How to use the application?

The application is very simple and intuitive to use. Everyone could easily learn how to play the game

The first window that appears in your computer is the initial window of the application. In this window, you can enter your user name or create a new one (in case you are not logged as an user)

Create a new user

If you want to create a new user, you must provide your name, your personal email and a password. All this information is required to create the user.

Play the game

To play to our game, you just have to press the dice and decide which category you want to play. Depending on the category selected, one question or another will be retrieved from the database and asked to the player.
