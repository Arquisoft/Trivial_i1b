What's Trivial_i1b about?

Trivial_i1b is a project for Software Architecture course in Computer Engineering degree.

The main goal of the project is to develop a Trivial game, following the indications gaved to us. It is based on the Trivial Pursuit game, the typical game based on a board and with different questions that belong into different categories to be answered by the players.

The desktop application (from the second deliverable) is developed in Java, using Eclipse, making use of the Maven plugin and different testing tools to test the correct performance of the game.

For the third deliverable, a web application developed with JavaScript and HTML

DESKTOP APPLICATION

This application is very simple and intuitive to use, since it is based in the original Trivial game. Everyone of any age can easily learn how to use it.

The interface of the game consists of 3 windows:

  • Initial window

    The first window that appears when the application is run. In this window, you can enter your user name or create a new one, in case you are not logged in as an user.

  • About

    In this window you can take a look at the information about the development team

  • Create a new user

    If you want to create a new user, you must provide some personal data like your name, your email and a password. All this information is required to create the user.

  • Play the game

    This is the main window of the application. Here is the board to show the player the different cells that can be selected.To start the game, the dice has to be pressed and the player has to decide which category he/she prefers. Depending on the category selected, one question or another will be retrieved from the database and asked to the player. The player has also the possibility of changin teh background colour of the window, selecting it with some buttons. The colours are those corresponding to the 6 different categories of the questions.

WEB APPLICATION

The web application has a very simple interface. To use it, users will have to Register as a new username, or to Log In if they have already created one.

    Interface:

  • Initial window

    • The first window that appears when the application is opened. In this window, you have two buttons:

  • Create new user: to register as new user in the game.
  • Log in: to log in with your username and your password (you must be registered to have accesto the game).

    • In the initial window, you can also change the background color of the application if you want to have another color as background to enjoy the game. You have several options (the colours of the trivial game):

  • Red
  • Blue
  • Yellow
  • Purple
  • Green

  • Create a new user

    If you want to create a new user, you must provide some personal data like your name, your email and a password. All this information is required to create the user.

    Once the user is created, you directly acces to the initial window again, to log in an starting playing the game.

  • Play the game

    This is the main window of the application. Here is the board to show the player the different cells that can be selected.To start the game, the dice has to be pressed and the player has to decide which category he/she prefers. Depending on the category selected, one question or another will be retrieved from the database and asked to the player. The player has also the possibility of changin teh background colour of the window, selecting it with some buttons. The colours are those corresponding to the 6 different categories of the questions.

The WEB interface is different to the Desktop application one, but the different options in each window are almost the same, so it makes the application to increaise in usability.
